# Yogyata Portal Frontend  

**Yogyata Portal** is an AI-driven platform for employers to streamline the hiring process with features like resume shortlisting and automated examination management.  
This repository contains the **frontend** implementation using **React**, **Vite**, and **Tailwind CSS**.  

---

## Features  
- **Employer Dashboard**: Manage job profiles, applications, and examinations.  
- **AI-Powered Resume Shortlisting**: Automatically shortlist candidates based on job descriptions.  
- **Automated Exam Portal**: Schedule, manage, and evaluate online examinations effortlessly.  

---

## Prerequisites  
Ensure you have the following installed on your machine before proceeding:  
- **Node.js** (v16 or higher)  
- **Git**  

---

## Getting Started  

### 1. Clone the Repository  
To clone this repository, run:  
```bash  
git clone https://github.com/acm-lucky/Yogyata_Portal_Frontend.git  
cd Yogyata_Portal_Frontend  
```
### 2. Install Dependencies
Install all required dependencies using npm:
```bash  
npm install  
```
### 3. Start the Development Server
Launch the development server with the following command:
```bash  
npm run dev  
```
The application will be accessible at http://localhost:5173.
